# Regular Shapes
![demo](demo.gif)

N-sided shapes rendered using WGPU.

## Controls
- `left`: decrement sides by 1.
- `right`: increment sides by 1.
- `down`: go to triangle.
- `up`: go to shape with highest sides.

## Usage
- clone the repo and `cd` into it.
- `cargo run --release`
